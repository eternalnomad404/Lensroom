## Installation

1. Clone the repository:
  ```bash
  git clone https://github.com/eternalnomad404/Lensroom.git
  ```
2. Navigate to the project directory:
  ```bash
  cd Lensroom
  ```
3. Install dependencies:
  ```bash
  npm install
  ```
4. Start the application:
  ```bash
  npm run dev
  ```
